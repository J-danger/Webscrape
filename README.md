# Webscrape

A web application that lets users view and leave comments on the latest Coindesk.com articles. 

Deployed link: https://frightening-eyeballs-04450.herokuapp.com/



Technologies used

   1. express

   2. express-handlebars

   3. mongoose

   4. cheerio

   5. axios

Future Development
